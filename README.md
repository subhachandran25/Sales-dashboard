# Sales Performance & Hierarchy Tracker

This application is an automated sales performance dashboard designed to track KPIs across a multi-tier organizational hierarchy (Area Head -> Senior Manager -> Sales Manager -> Sales Rep).

## 🚀 Features
- **Hierarchical Filtering:** Drill down from Region to individual Sales Rep performance.
- **KPI Tracking:** Monitor Conversion Rates, Call Efficiency, and Lead Management.
- **Automated Reporting:** Real-time visualization of sales data.
- **Performance Metrics:**
    - **Conversion Ratio:** (Converted Leads / New Leads)
    - **Call Efficiency:** (Deals Closed / Total Calls)
    - **Activity Volume:** Tracking total calls and call duration.

## 📂 Project Structure
- `app.py`: The main Streamlit application code.
- `data.csv`: The dataset containing the sales hierarchy and performance metrics.
- `generate_data.py`: Script used to generate the hierarchical dummy data.
- `requirements.txt`: List of dependencies.
